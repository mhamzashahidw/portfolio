# 🌐 HamzaDevLog: Personal Portfolio & Technical Showcase

Welcome to my personal developer hub and open-source ledger. This repository hosts the architecture, modular layouts, and client-side logic for my personal portfolio platform, deployed live via GitHub Pages. It acts as my primary digital footprint, aggregating software engineering milestones, academic triumphs, and foundational code implementations.

🚀 **Live Deployment:** Explore the interactive platform live at [mhamzashahidw.github.io/hamzadevlog/](https://mhamzashahidw.github.io/hamzadevlog/)

---

## 🛠️ Tech Stack & Engineering Core

- **HTML5 (`index.html`):** Semantic layout structure mapping clean navigation nodes, responsive grid containers, technical summaries, and structured project modules.
- **CSS3 (`style.css`):** Comprehensive global style guide tracking typography scaling, custom flexbox grids, fluid card alignments, and dynamic interactive hover transitions.
- **JavaScript (`script.js`):** Client-side scripts driving active element mutations, component behaviors, or interactive component control layers.
- **Asset Hub Optimization:** Structured asset routing pointing to a relative media tree with optimized image containers to ensure zero layout shifts and rapid page load velocities.
